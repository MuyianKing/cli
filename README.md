## Install

```bash
# 首先进行全局安装
npm i @muyianking/cli -g

# yarn
yarn global add @muyianking/cli

# pnpm
pnpm i @muyianking/cli -g
```

### Usage

```bash
# 创建web项目
mu web

# 创建h5项目
mu h5

# 创建简易html项目
mu html
```
